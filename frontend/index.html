About the Engine

The Hometown Success Engine is a hackathon prototype that maps where America's elite athletes actually come from. We clustered the hometowns of 5,012 Olympians and Paralympians (2020–2024) into 37 regional hubs using HDBSCAN, then surfaced the 5 regions where Paralympic representation runs more than double the national rate.

Each hub combines real roster data with Gemini-generated narratives that explain its athletic identity. The chat assistant uses Gemini Tool Use to drive the map directly: ask about a region or hub, and it filters and zooms for you.

Built for the Google Cloud × Team USA Challenge. Imagined as a tool for USOPC scouts and youth-program leaders to spot under-represented hometowns where the next Team USA roster could emerge.

Source: Team USA roster 2020–2024 · Built with Gemini, Google Maps, deck.gl, and Cloud Run
